1

Prepare the remote control

1.Long press st-c1 power switch, the green light is on.
2.Expand the st-c1 antenna.
3.Install the bracket onto the st-c1 bayonet.
2

Turn on the OTG function of mobile devices (take the mobile phone as an example here)

1.Click the "Settings" button, enter the "system" - "about the phone" - "software information", click the "version number" for 5 times in a row, and call out the developer option；
2.Click the "Settings" button and go to "system" -- "developer options" -- turn on the "USB debugging" switch；
3

Connect mobile devices

1.Attach the mobile device to the remote control bracket；
2.Use the standard OTG data line to connect the mobile device to st-c1. For non-type-c mobile devices, please use the standard Micro USB adapter. Apple devices can use the original apple data cable.
3.Toggle switch according to phone type.
4.When you open the Senseplay APP, you need to activate st-c1 for the first time. Please complete the activation according to the APP prompt. The activation code is 6 digits after the SN code is labeled on the st-c1 body. SN:XXXXXXXXXXXXXXXXXX- XXXXXX
4

Connect FPV unmanned vehicle

1.Insert the smart battery into the battery slot of the FPV unmanned vehicle, and press the battery switch for a long time. When the green light is on, it will be turned on.
2.Senseplay APP will automatically search for nearby devices.
3.After success, select the FPV unmanned vehicle to be connected and click "ok" button
4.FPV unmanned vehicle needs to be activated for the first time. Please complete the activation according to the APP. The activation code is the 6 digits after the SN code is labeled on the bottom of the FPV unmanned vehicle. SN:XXXXXXXXXXXXXXXXXX- XXXXXX
5.When activated click,enter the first view operation interface.
